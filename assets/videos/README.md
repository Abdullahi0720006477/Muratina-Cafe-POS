# Login Background Videos

Drop three short, muted, looping clips here and the login page will play them
as a cross-fading cinematic background (with a rotating caption):

| File name   | Suggested footage                          | Caption shown                 |
|-------------|--------------------------------------------|-------------------------------|
| `coffee.mp4`| Espresso / latte pour, beans, brewing      | "Freshly brewed coffee"       |
| `juice.mp4` | Fresh fruit, juicing, smoothies            | "Cold-pressed fruit juices"   |
| `meals.mp4` | Plated meals, cooking, serving             | "Delicious meals, served fresh" |

## Tips
- Keep each clip **5–15 s**, **muted**, **1080p**, and **MP4 (H.264)**.
- Aim for **< 5 MB** per file so the login loads fast.
- If a file is missing, the page automatically falls back to a poster image —
  so the login always looks good even before you add the videos.

To change the captions or filenames, edit the `.bg-scene` blocks in `index.php`.
